# Exécuter et déployer votre application AI Studio

Cela contient tout ce dont vous avez besoin pour exécuter votre application localement.

Voir votre application dans AI Studio : https://ai.studio/apps/temp/1

## Exécuter localement

**Prérequis :** Node.js

1. Installer les dépendances :
   `npm install`
2. Définir la `GEMINI_API_KEY` dans [.env.local](.env.local) avec votre clé API Gemini
3. Exécuter l'application :
   `npm run dev`
